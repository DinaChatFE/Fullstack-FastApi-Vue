## How to install

## in the source files it contians requirements.txt
if you have vurtual environment installed
command
```bash
    pip3 install -r requirements.txt
```
if not, install virtual environment in python via
```bash
  python3 -m venv venv
```
go inside enviroment by command
```bash
  . venv/bin/activate
 ```
